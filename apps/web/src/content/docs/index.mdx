## Quick Start

```bash
# Install OpenGenerator
npm install opengenerator

# Initialize a new project
npx opengenerator init

# Generate your API
npx opengenerator generate
```

## Supported Technologies

### Schema Parsers
- **Prisma** - Parse Prisma schema files
- **OpenAPI** - Parse OpenAPI 3.x specifications
- **JSON Schema** - Parse JSON Schema definitions
- **Zod** - Parse Zod schema definitions
- **TypeBox** - Parse TypeBox schemas
- **VLD** - Parse VLD validation schemas

### Framework Adapters
- **Express** - Classic Node.js framework
- **Fastify** - High-performance framework
- **Hono** - Ultrafast edge framework
- **Koa** - Next-generation Express
- **Standalone** - Zero dependencies

### Authentication
- **JWT** - JSON Web Token authentication
- **OAuth** - OAuth 2.0 providers (Google, GitHub, etc.)
- **Session** - Cookie-based sessions
- **API Key** - API key authentication
- **Magic Link** - Passwordless email login
- **Passkey** - WebAuthn/FIDO2 authentication

### Database Adapters
- **Prisma** - Type-safe ORM
- **Drizzle** - Lightweight TypeScript ORM
- **Kysely** - Type-safe SQL query builder
- **TypeORM** - Full-featured ORM
- **Mongoose** - MongoDB ODM
- **Raw SQL** - Direct SQL queries

### Deployment
- **Docker** - Container deployment
- **Vercel** - Serverless deployment
- **Railway** - Platform deployment
- **Fly.io** - Edge deployment
- **AWS Lambda** - Serverless functions
- **Kubernetes** - Container orchestration

## Example Configuration

```typescript
import { defineConfig } from 'opengenerator'

export default defineConfig({
  schema: './prisma/schema.prisma',
  output: './generated',

  api: {
    rest: { enabled: true, prefix: '/api/v1' },
    graphql: { enabled: true },
  },

  adapter: 'fastify',
  auth: { strategies: ['jwt'] },
  database: { adapter: 'prisma' },
  deploy: ['docker'],
})
```

## Presets for Common Use Cases

- **fullstack-prisma** - Full-stack with Prisma + REST + GraphQL + Fastify
- **serverless-drizzle** - Serverless with Drizzle + REST + Hono + Vercel
- **graphql-apollo** - GraphQL-first with Apollo Server
- **minimal-rest** - Minimal REST API with API key auth
- **enterprise** - All features enabled for enterprise apps
